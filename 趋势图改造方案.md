# 趋势图改造方案（A + C + 时间切换）

> 目标：让"谁变了、变了多少、哪天变的"一眼可见，解决当前对数轴+绝对价格"平行线看不出变动"的问题。

## 问题根因（先对齐）
1. **对数轴压平小变化**：为了让 $0.24~$180 同屏用了对数轴，代价是 3%~4% 的真实调价在对数刻度上几乎看不见。
2. **多数模型确实没动**：12 天里只有智谱、MiniMax 等个别家微调过，其余本就是平线。
所以方案核心 = 把"变动"本身做成主角，绝对价格退居明细表。

---

## 改造内容

### ① 趋势主图加「绝对价 / 归一化指数」开关（方案A，核心）
- 主图右上角新增一个切换：`绝对价格` ｜ `归一化指数`，**默认归一化指数**。
- **归一化指数**：每个模型取所选时间窗口第 1 天价格 = 基准 100，之后画相对值。
  - 没调价 → 一条贴着 100 的平线；降价 4% → 明显掉到 96；涨价 10% → 升到 110。
  - 纵轴改成线性（指数模式下不需要对数轴），Y 轴标注"指数(首日=100)"。
  - tooltip 显示：`模型名 指数 96.0（较首日 ▼4.0%）`。
- **绝对价格**：保留现在的对数轴视图（看绝对贵贱），与指数视图一键互换。
- 「输入价/输出价」开关保留，和上面两种模式自由组合。

### ② 近期变动提示带（方案C）
- 趋势图**正上方**加一条窄提示带，自动扫描当前时间窗口内有调价的模型：
  - 有变动：`近 7 天调价：智谱 glm-5.2 输入 ▼3.1% ｜ MiniMax m2.7 输入 ▼4.0%`（涨红降绿小标签）。
  - 无变动：显示淡灰文字"近 7 天 16 个模型价格均无变动"。
- 让用户不用盯图也能立刻知道发生了什么。

### ③ 时间范围切换（方案B，顺带做）
- 主图右上角加：`近7天` ｜ `近14天` ｜ `全部` 三个按钮，**默认近7天**。
- 切换只影响趋势图与变动提示带的时间窗口；指数基准也随窗口首日重算。
- 数据不足 7 天时按钮自动按实际天数显示。

---

## 交互布局（趋势卡片自上而下）
```
价格趋势                          [近7天|近14天|全部] [指数|绝对价] [输入|输出]
近 7 天调价：智谱 glm-5.2 ▼3.1% ｜ MiniMax m2.7 ▼4.0%   ← 变动提示带
[全部厂商] [OpenAI] [Anthropic] … [豆包]                ← 厂商筛选(已有)
┌────────────────────────────────────────────┐
│  指数折线图（默认，首日=100，看变动）          │
└────────────────────────────────────────────┘
```

## 不改动的部分
- 厂商筛选 chips（已修为简称）、KPI、旗舰vs次旗舰、散点、综合排行、明细表均不变。
- 仍单文件 ECharts CDN、数据前端渲染、浅色商务风。

## 默认值汇总（可调）
- 时间范围：**近7天**　｜　视图模式：**归一化指数**　｜　价格口径：**输入价**
- 这三个默认就是"最容易看出变动"的组合；用户可随时切回绝对价/全部时间。
